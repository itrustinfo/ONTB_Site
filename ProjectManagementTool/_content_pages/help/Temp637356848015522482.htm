1.1  View  Documents:

Click “Documents” in main menu bar. You see  following options.

 

 

Click “View Documents” you will see following screen.

 

 

There is a option to view by “category” or by “activity”.

a.       “by Category”: The documents are assigned to various categories like civil engineering documents/Structural drawings etc.

b.       or “by Activity”: The documents can be associated with an activity.

 

1.1.1        View  Documents by “category”:

If you click  “by category “, You will see list of categories in the tree.

Choose any category whose documents you want to view. You will see following screen.

 

 

1.                     For each submittal

i.                       Name of the Submittal,

ii.                      Activity name (if submittal is associated with an activity”

iii.                    You can see date it was added.

iv.                    A link to add documents to the submittal.

v.                      Edit properties of the submittal.

vi.                    Delete the submittal.

5.1.2Adding a submittal:      The Submittals are generally added at the time of project initiation.

a.      Click “Documents” in main menu bar.

b.      Click the work package for which submittal belongs

c.       Choose the submittal and click “add submittal” and you will see following screen.

 

The fields are shown below.

a.      Document Name: Name of the submittal.

b.      Document category: A drop down list of pre-defined categories. Choose one.

c.       Document flow: This defines a process a submittal flows through. This includes

1.      Steps and the sequence of steps (Example: Submission, review, approval)

2.      Users who are responsible for each step to be completed.

3.      Suggested target date.


 

2.      If you have clocked “by activity“,you will see following screen. Click the activity you want to see

3.      You will see list of submittals under that category.

4.      For each submittal

vii.               You can see date it was added.

viii.             A link to add documents to the submittal.

ix.                 Edit properties of the submittal.

x.                   Delete the submittal

 

 

 

5.1.3Deleting a submittal:

 

a.      Click “Documents” in main menu bar.

b.      Click “View Documents”.

c.       Click “By category”

d.      Click the work package for which submittal belongs

e.      Click the delete (thrash symbol) in the row which shows submittal.

You will see following screen.

 

 

Click “OK”.

The corresponding submittal will be removed from the list.

1.2  Submitting a document (first part of a flow generally):

 

a.      Click “Documents” in main menu bar.

b.      Click “By category”

c.       Click the category for which submittal belongs

d.      Choose the submittal and click “add documents” and you will see following screen.

 

The fields are

a.      Subject/Description: Any description for the submittal that would help describe it.

b.      Reference number: This is system generated reference number to identify the submission for NJSE documents. For Contractor and BWSSB options, this field is entered based on the letter received.

c.       Document media: Select the media on which the document was submitted.

d.      File reference number: This is the reference number used in current system where the correspondence is filed.

e.      Remarks: Any additional noting while uploading.

f.        Document type: Choose if the document is a cover letter or a general document.

g.       Originator: Source of the document (Contractor/BWSSB/NJSE)

h.      Originator reference number: The reference number in the cover letter received.

i.        Incoming receive date: date on which the cover letter and documents were received.

j.        Cover letter date: date the cover letter is dated.

k.       Choose file (cover letter): file that has scanned copy of cover letter.

l.        File or folder: There can be a single document or multiple documents in a folder. Choose appropriate option.

m.    Choose file or folder: Browse the local system and select the file or folder to be uploaded.

n.      Click “Submit” to submit the document.

e.      On the document whose status you want to update click on “view history”.

f.        You will see following screen.

 

 

g.       Click on “Add status”. You will see following screen.

 

 

 

 

h.      The “status” shows a drop down of possible states you can move the documents (based on the flow process) and user.

i.        Choose a status.

j.        Add a reference number .

k.       Enter date of cover letter.

l.        Choose actual date.

m.    Browse and select scanned copy of cover letter.

n.      Browse and select “review file” (the file that contains review feedback, if any).

You will see following screen which shows the document had been updated.
