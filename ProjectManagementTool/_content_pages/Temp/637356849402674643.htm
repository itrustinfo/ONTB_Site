1.      Issues and concerns:This feature allows you to add any issues and concerns that need attention.

 

 

      To use this functionality,

a.      Choose “Miscellaneous” in main menu.

b.      Click on “Issues”.

c.       You will see following screen.

 

 

This shows details of issues flagged for any chosen task/sub-task.

1.      Total no of issues,

2.      Open issues,

3.      Closed issues,

4.      Rejected issues

5.      Table describing issues

a.      Serial number,

b.      Description,

c.       Date of issue occurrence

d.      Proposed closure date

e.      Actual closure date

f.        Status

g.       remarks (for latest status).

 

 

9.1 Add Issues:

 

a.      Choose “Miscellaneous” in main menu.

b.      Click on “Issues”.

c.       In the displayed screen (shown above), Click “Add Issues”.

d.      You will see following screen.

 

Enter the following data:

a.      Activity Name,

b.      Issue description,

c.       Assigned user : Choose from option provided which user you want to assign this issue to.

d.      Assigned date: Date of assignment.

e.      Reporting user : Choose from option provided which user reported the issue.

f.        Reporting date: Date of reporting.

g.       Approving user : Choose from option provided which user can approve closure when issue is closed.

h.      Approving date: Date of reporting.

i.        Proposed date of closure.

j.        Current status (when entered this should be “Open”).

k.       Remarks (if any) .

l.        Click Submit.
