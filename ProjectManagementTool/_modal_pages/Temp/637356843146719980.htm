1.0  Masters:.

 

Masters are master information which will be added to the system to be used in subsequent stages to create projects/workpackagesetc…This is to view/update master data about contractors, work packages etc...Creation of masters is first step to configuring the application. This is restricted to Application administrators. Following paragraphs explain creation of masters.

 

Click on “Masters” in main menu.You will see following screen.

 

 

 

    The options Projects, Contractors, Work packages help you add  these details to the masters.

1.1.1        Adding Project Categories to the master.

a.      Click on “Masters” in main menu.

b.      Click on “Projects”.

c.       You will see following screen

 

     Click on “Add project categories”. You will see following screen.

 

 

           Enter

a.      Project Category ,

b.      Description

c.       Click “submit”

1.1.2        Adding Projects to the master.

 

a.      Click on “Masters” in main menu.

b.      Click on “Projects”.

c.       Click on “Add projects”

d.      You will see following screen

 

           Enter

a.      Select Project category,

b.      Add project name,

c.       Add Project code,

d.      Add Funding Agency,

e.      Add start date,

f.        Select Currency (INR/USD/YEN)

g.       Add Budget

h.      Add Actual expenditure (if project has already started)

i.        Add Planned end date,

j.        Add Projected end date.

         Click “Submit”

 

1.2  Adding contractor.

 

 

a.      Click on “Masters” in main menu.

b.      Click on “Contractors”.

c.       You will see following screen

 

This shows the list of contractors already added.

 

Click on “Add contractor”. You will see following screen.

 

Add following details.

a.      Name

b.      Type of contract.

c.       Contractor representative,

d.      Contract value,

e.      Contract duration,

f.        Agreement date,

g.       Start date,

h.      Letter of acceptance date,

i.        Completion date.

j.        Click “submit”.

This will get added to the list of contractors.

 

1.3  Adding Work packages related data to the master.

Following lists are added under this class.

1.      Work Packages,

2.      Location master and

3.      Client master.

You click on Masters,

Click on “work packages”

You will see following screen

 

 

 

 

1.3.1        Adding Workpackages names to the master.

 

Click on “Add workpackage master”

 You will see following screen.

 

 

1.      Choose the project of interest.

2.      Enter work package name,

3.      Enter Workpackage code.

4.      Click “submit”. This will update the master record for work package.

 

1.3.2        Adding Location names to the master.

 

a.      Click on “Masters” in main menu.

b.      Click on “Workpackages”

c.       Click on “Add Location master”.

You will see following screen.

 

 

a.      Choose the project,

b.      Enter location,

c.       Enter location code

d.      Click “submit”
